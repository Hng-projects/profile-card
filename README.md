# Profile Card (HNG Stage 1B Task)

A responsive, accessible, and interactive profile card built with semantic HTML, vanilla CSS, and vanilla JavaScript. This project satisfies the requirements for Stage 1B of the HNG Internship.

## Setup

Simply open `index.html` in your favorite web browser. No build steps or complex dependencies required!
